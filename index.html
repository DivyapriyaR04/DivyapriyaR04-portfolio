ASL Alphabet Recognition: Python CV & ML Project

The Sign Language Detection project employs Python, integrating key packages such as Mediapipe, OpenCV-Python, NumPy, Scikit-learn, and modules like Pillow and OS. Utilizing computer vision, it captures real-time hand movements through OpenCV and Mediapipe for hand tracking and feature extraction. NumPy handles numerical operations, while Pillow aids image processing, and OS manages file operations. The system utilizes the Random Forest classifier from Scikit-learn to categorize hand gestures. The translated gestures are displayed as text on the webcam feed, enhancing accessibility by providing real-time interpretation of sign language into readable text. This interdisciplinary approach amalgamates computer vision, machine learning, and user interface for efficient sign language recognition.
